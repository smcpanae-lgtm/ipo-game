# The IPO Path: 栄光への決断 — CLAUDE.md

## プロジェクト概要
IPO準備の専門知識を自然に学べるCLIシミュレーションゲーム。
日本公認会計士協会「新規上場のための事前準備ガイドブック」の実務的指摘事項をゲームギミックとして組み込む。

## ディレクトリ構造
```
ipo_game/
├── main.py          # ゲームエントリーポイント・上場審査・エンディング
├── models/
│   ├── company.py   # 会社・ゲーム状態データクラス
│   └── events.py    # イベント・フラグデータクラス
├── engine/
│   ├── timeline.py  # N期管理・四半期進行・定時株主総会
│   ├── finance.py   # キャップテーブル・収益認識・CF計算
│   └── roulette.py  # 確率ルーレット・因果応報フラグ管理
├── scenario/
│   └── ipo_knowledge.py  # IPO実務イベントライブラリ（13+種別）
└── ui/
    └── display.py   # richダッシュボード・入力UI
```

## IPO学習ロードマップ
| フェーズ | 期間 | 主要テーマ |
|---------|------|-----------|
| N-3期 | 体制構築期 | ショートレビュー・管理体制の基礎構築 |
| N-2期 | 直前々期 | 監査契約・発生主義移行・棚卸立会 |
| N-1期 | 直前期 | 内部統制報告制度・コンプライアンス強化 |
| N期   | 申請期 | 上場審査・東証形式要件・実質審査 |

## ゲームシステム設計
- 1ターン = 1四半期（Q）
- 各Qに選択肢イベント1〜2個
- 選択 → ルーレット（確率）→ 即時結果 + 将来フラグ
- 隠れリスク（爆弾）：ショートレビューで可視化
- 因果応報：N-3期の手抜きがN期の上場否認を招く

## 主要イベントカテゴリ（公認会計士協会ガイドブック準拠）
1. 証憑管理の未整備
2. 発生主義・収益認識基準への対応
3. 棚卸資産管理（実地棚卸立会）
4. 原価計算体制
5. 資産・負債の管理
6. 連結決算体制
7. 関連当事者取引の整理
8. 内部管理体制（職務分掌・出納と記帳の分離）
9. 労務管理（未払残業代）
10. 情報システム内部統制
11. 不正防止（横領リスク）
12. 会計上の見積り
13. 反社会的勢力排除チェック
14. 監査法人選定・監査契約
15. 資本政策・キャップテーブル管理
